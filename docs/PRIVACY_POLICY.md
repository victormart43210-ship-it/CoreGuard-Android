# CoreGuard Privacy Policy

> Effective: July 2026  
> Package: `com.coldboar.coreguard`  
> Host this document at a public HTTPS URL and paste that URL into Play Console.

## Summary

CoreGuard is a **prototype** Android app for on-device memory display, local
security-check heuristics, and an educational Network Defense Lab simulation.

**We do not operate a user account system and do not sell personal data.**

## Data we collect

| Category | Collected? | Notes |
|----------|------------|-------|
| Name, email, account | No | No login |
| Location | No | — |
| Contacts / photos | No | — |
| Device identifiers for ads | No | No ads SDK |
| Purchase / payment data | No on this build | Premium unlock is a **local demo** only — not Google Play Billing |
| Crash / analytics | No | Not integrated |
| Network Defense Lab state | On-device only | In-memory simulation; cleared when the process ends |

## Permissions

This build declares **no dangerous runtime permissions**. RAM usage is read via
`ActivityManager` APIs available to the app process.

## Network

This prototype does **not** require Internet permission for core features. If a
future release adds purchase verification or an API, this policy will be updated
and Data Safety answers revised before that version ships.

## Children’s privacy

CoreGuard is not directed at children under 13. We do not knowingly collect
children’s personal information.

## Security claims

Security dashboard results and the Network Defense Lab are **heuristic /
educational**. They are not a guarantee against malware, intrusion, or device
compromise. Do not interpret store listing or in-app text as “spyware,” hacking,
or undetectable monitoring capabilities — CoreGuard does not provide those.

## Contact

For privacy questions about this prototype, contact the repository maintainer via
the GitHub project for `CoreGuard-Android`.

## Changes

We may update this policy when features change. Material changes will be
reflected in a new Effective date above and in Play Console Data Safety.
