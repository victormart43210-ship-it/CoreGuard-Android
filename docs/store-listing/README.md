
(Character count: 70)

## Full description

```
CoreGuard helps you understand your Android device’s security posture with local, on-device checks — no account required.

What you get
• RAM usage from the system ActivityManager
• A Security Dashboard with heuristic checks (debugger, root indicators, emulator, build type, signature)
• Network Defense Lab — an educational 16-node attack/defense simulation (BFS/DFS, defenses, MST guidance)
• Honest labeling when a metric is simulated (CPU is not measured in this prototype)

What this is not
• Not a live intrusion-prevention or antivirus product
• Not spyware, remote monitoring, or undetectable tracking
• Premium unlock in this build is a local demo only — not a Google Play purchase

Privacy
CoreGuard does not require a login and does not sell personal data. See the in-console privacy policy URL for details.

Prototype status
Security heuristics can be bypassed on advanced rooted devices. Google Play approval is never guaranteed by uploading a build.
```
